Welcome to the Flashcord Store Template!

Thanks to FlashCFG, you can set up your very nice store page for your module, plugin or theme in an instant!

Don't forget however that this page is 100% fully customizable by yourself manually, yes you can put whatever you want if you're not satisfied with the default layout!

If you're an advanced person, you may take a look on how SNDL works and how to use it to make absolutely dazzling pages.

Note that SNDL is currently not or barely documented, you shouldn't have to do this but you can.
